# 🏨 Royal Stay Hotel Website

A modern and responsive hotel website built using **HTML, CSS, and JavaScript**. This project showcases a luxury hotel with elegant design, room listings, facilities, a restaurant, a gallery, and a contact page.

## 🚀 How to Run the Project

1. Download or clone this repository.
2. Extract the project folder (if downloaded as a ZIP).
3. Open the project folder in Visual Studio Code.
4. Open `index.html` in your browser, or use the **Live Server** extension for a better development experience.
